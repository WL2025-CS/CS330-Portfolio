# CS330-Portfolio
### Final 3D Scene Project Reflection

### How do I approach designing software?

I approach software design by first identifying the core objective and breaking it into manageable components. For this 3D scene, I began by planning object placement, camera perspective, and lighting structure before writing code. I focused on spatial organization and visual balance to create a cohesive environment. This structured planning process helped ensure that each object contributed meaningfully to the final composition.

### What new design skills did this project help me develop?

This project strengthened my understanding of scene composition, texture mapping, lighting configuration, and spatial transformations. I developed stronger skills in organizing objects within a 3D coordinate system and balancing visual realism with performance considerations.

### What design process did I follow?

I followed an iterative design process. I began with basic geometric shapes, gradually refined object detail, applied textures, adjusted lighting, and fine-tuned camera positioning. Each milestone built upon the previous version, allowing continuous refinement and improvement.

### How do I approach developing programs?

When developing programs, I focus on modularity and incremental testing. I separated rendering logic, transformations, and texture application into organized sections. Testing small changes frequently helped prevent larger debugging issues later.

### How did iteration factor into development?

Iteration was central to this project. Each milestone introduced improvements to lighting, materials, and object realism. Repeated adjustments to textures and camera positioning significantly enhanced the final visual output.

### How has my development approach evolved?

Throughout the milestones, my coding became more structured and intentional. I became more comfortable working with OpenGL concepts such as transformations, texture coordinates, and lighting calculations. My confidence in debugging graphical output also improved significantly.

### How can computer science help me reach my goals?

Computer science equips me with problem-solving skills, logical reasoning, and technical proficiency. These skills are foundational for building scalable systems and developing innovative solutions across multiple domains.

### How do computational graphics and visualization support my future pathway?

Computational graphics and visualization enhance my understanding of spatial reasoning, rendering pipelines, and visual simulation. These skills are applicable in areas such as software development, simulation modeling, game development, data visualization, and user interface design. Professionally, they strengthen my ability to create visually engaging and technically sound applications.
